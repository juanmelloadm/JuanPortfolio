# 💻 Portfólio Profissional - Juan Pablo

Este projeto é meu portfólio pessoal desenvolvido com foco na área de Tecnologia da Informação.  
A proposta foi criar um site moderno e objetivo para apresentar meus conhecimentos, formação e interesses dentro do universo de TI.

---

## 🚀 Sobre o projeto

Este portfólio foi desenvolvido como uma forma de consolidar meus conhecimentos em desenvolvimento web e também servir como uma vitrine profissional.

A interface foi pensada para ser simples, direta e com um visual atual, destacando:

- Minha apresentação profissional
- Áreas de interesse em TI (suporte, redes e segurança)
- Formação acadêmica
- Cursos realizados
- Objetivos profissionais

---

## 🛠️ Tecnologias utilizadas

Durante o desenvolvimento, utilizei as seguintes tecnologias:

- **React** → construção da interface
- **Vite** → ambiente de desenvolvimento rápido
- **Tailwind CSS** → estilização moderna e responsiva
- **Git & GitHub** → versionamento de código
- **Vercel** → deploy e hospedagem do projeto

---

## 🎯 Desafios enfrentados

Durante o desenvolvimento, enfrentei alguns desafios importantes que contribuíram bastante para meu aprendizado:

### 🔹 Configuração do ambiente
Tive que lidar com:
- instalação e configuração do Node.js
- ajustes de permissões no PowerShell
- entendimento do funcionamento do npm

---

### 🔹 Integração com Tailwind CSS
A configuração do Tailwind exigiu atenção, principalmente por conta das mudanças nas versões mais recentes.

Foi necessário:
- configurar corretamente o Vite
- entender a aplicação das classes utilitárias
- ajustar problemas visuais iniciais

---

### 🔹 Estruturação do projeto
Aprendi a organizar melhor um projeto front-end, entendendo:
- separação de arquivos
- organização de código
- boas práticas iniciais com React

---

### 🔹 Uso do Git
Durante o processo, pratiquei:
- criação de repositórios
- versionamento de código
- resolução de erros comuns
- envio de projetos para o GitHub
